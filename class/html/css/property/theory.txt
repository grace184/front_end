******** border property 정리 ******

1. 4방향 모두 공통 적용
border: 굵기 종류 색;


2. 각각의 방향에 공통적용

border- top: 굵기 종류 색;
border- right: 굵기 종류 색;
border- bottom: 굵기 종류 색;
border- left: 굵기 종류 색;

3. 각각의 방향, 각각의 CSS 속성 적용

border-top-style,
border-top-width,
border-top-color,
border-right-style,
border-right-width,
border-right-color,
...
